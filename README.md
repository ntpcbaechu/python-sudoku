# 🧩 Python Console Sudoku Game (主控台數獨小遊戲)

這是一個使用純 Python 打造的基底數獨遊戲，玩家可以在終端機（Terminal / CMD）進行互動與解題。

## ✨ 專案特點
* **二維列表資料結構**：使用 9x9 陣列記錄數獨棋盤與狀態。
* **即時規則判斷**：自動檢查玩家填入的數字是否符合同行、同列與 3x3 九宮格規則。
* **友善介面**：視覺化分隔線與座標提示，方便輸入。

1. **確保電腦已安裝 Python 3**
